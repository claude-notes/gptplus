问答

预约前最常见的问题

Codex 是什么？

Codex 是 OpenAI 的智能编程代理，可以在获得授权后读取项目文件、修改代码、运行命令并完成测试。

Codex 怎么安装？

在 Windows 或 macOS 上安装并打开 ChatGPT 桌面应用，登录 ChatGPT 账号，选择 Codex，授权需要使用的项目文件夹，再完成一次测试任务。

Windows 和 macOS 都能安装 Codex 吗？

可以。OpenAI 当前的 ChatGPT 桌面应用在 Windows 和 macOS 上提供 Codex，具体功能与系统要求以官方最新说明为准。

手机上可以独立运行 Codex 吗？

手机端主要通过 ChatGPT 移动应用远程查看、批准和继续受支持的桌面 Codex 任务，不等同于在手机本地独立运行桌面 Codex。

完全零基础可以使用吗？

可以。基础安装包适合先完成电脑端安装和首次测试；需要手机远程配置和真实项目陪跑时可选择标准跑通包。

一定需要 ChatGPT Plus 吗？

不一定。OpenAI 的 Codex 使用资格和用量会随 ChatGPT 计划变化，实际以官方说明为准；本服务套餐默认包含 1 个月 Plus 等值权益。

一般需要多久？

基础安装通常需要 45-90 分钟，标准跑通包一般预留 90-150 分钟，具体取决于设备、账号和网络环境。

基础安装包现在多少钱？

基础安装包当前优惠价为 499 元，原价 599 元，包含 1 台 Windows 或 macOS 设备远程安装、登录配置和一次测试任务。
